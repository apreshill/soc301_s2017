# But...what is a statistical graphic?

- A statistical graphic is a mapping of `data` variables to `aes()`thetic
attributes of `geom_`etric objects.

---

# A scatter-plot example

From `ggplot2movies` package, the `movies` data set:

```{r, echo=FALSE, message=FALSE, warning=FALSE, fig.height=5.7, fig.width=10}
library(ggplot2)
library(ggplot2movies)
# Scatterplot i.e. bivariate plot
ggplot(data = movies, aes(x=budget, y=rating)) +
  geom_point() +
  labs(x="Movie Budget", y="IMDB Rating", title="IMDB Rating vs Budget")
```

# A scatter-plot example - the code!

From `ggplot2movies` package, the [`movies`](ismayc.github.io/effective-Data-Storytelling-using-the-tidyverse/movies.html) data set:

```{r, message=FALSE, warning=FALSE, fig.height=3.5, fig.width=10}
library(ggplot2); library(ggplot2movies)
ggplot(data = movies, aes(x = budget, y = rating)) +
  geom_point() +
  labs(x = "Movie Budget", y = "IMDB Rating", 
       title = "IMDB Rating vs Budget")
```

# A line-graph example

From `nycflights13` package, the `flights` data set:

```{r, echo=FALSE, warning=FALSE, message=FALSE, fig.height=5, fig.width=10}
# Line graph
library(nycflights13)
library(dplyr)
library(lubridate)
flights %>% 
  mutate(date = ymd(paste(year, month, day, sep="-"))) %>% 
  filter(month==1) %>% 
  group_by(date) %>% 
  tally() %>% 
  ggplot(data=., aes(x=date, y=n)) + 
  labs(x="Date", y="Number of Flights", title="Number of Daily Flights from NYC in Jan 2013") +
  geom_line()
```

--

`geom()`etry is `geom_line()`
